# Glossário

**Array**  
Coleção de tamanho fixo cujos elementos possuem o mesmo tipo.

**Compilador**  
Programa que transforma código-fonte em código executável.

**Exportado**  
Nome iniciado por letra maiúscula e acessível por outros pacotes.

**Go Module**  
Unidade versionada de código Go definida por um arquivo `go.mod`.

**Map**  
Coleção de pares chave-valor com acesso direto pela chave.

**Package**  
Conjunto de arquivos Go no mesmo diretório e com o mesmo nome de pacote.

**Primary key lógica**  
Identificador único usado pelo projeto mesmo sem um banco de dados.

**Slice**  
Visão dinâmica sobre um array, usada como principal coleção sequencial em Go.

**Struct**  
Tipo que agrupa campos de tipos diferentes.

**UUID**  
Identificador de 128 bits usado para criar IDs com probabilidade extremamente baixa de colisão.

**Zero value**  
Valor padrão atribuído por Go a uma variável que não recebeu valor explícito.
