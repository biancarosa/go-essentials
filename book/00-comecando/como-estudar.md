# Como estudar

Use cada aula em quatro passos:

1. Leia a explicação e execute os exemplos.
2. Digite o código em vez de apenas copiar.
3. Evolua o Contact Book conforme a etapa do capítulo.
4. Responda às perguntas de prática sem consultar a explicação primeiro.

{% hint style="success" %}
Quebrar o programa faz parte do processo. Leia a mensagem do compilador, identifique o arquivo e a linha e tente explicar o erro antes de corrigi-lo.
{% endhint %}

## Convenções do livro

- A explicação está em português.
- Identificadores de código estão em inglês.
- Comandos de terminal aparecem em blocos `bash`.
- Arquivos Go aparecem em blocos `go`.
- Desafios extras são opcionais.

## Exemplos executáveis

O diretório [`examples/contact-book`](https://github.com/biancarosa/go-essentials/tree/main/examples/contact-book) contém um checkpoint para cada aula.

Entre no diretório correspondente e execute:

```bash
go run .
```

Quando houver testes:

```bash
go test ./...
```
